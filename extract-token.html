🔑 Twitter Auth Token Extractor

⚠️ Pourquoi le Bot Est Bloqué

Twitter détecte l'automatisation et affiche en boucle le champ email au lieu de passer au mot de passe. C'est une protection anti-bot.

Solution: Extraire manuellement le auth_token et le donner au bot.

📋 Étape 1: Se Connecter sur X.com

  1. Ouvrez un nouvel onglet
  2. Allez sur https://x.com
  3. Connectez-vous normalement avec email + mot de passe
  4. Vérifiez que vous voyez votre timeline

🔍 Étape 2: Extraire le Auth Token

Méthode 1 - Console JavaScript:

  1. Sur la page X.com, appuyez sur F12
  2. Cliquez sur l'onglet "Console"
  3. Copiez-collez ce code:

Méthode 2 - Via DevTools Application:

  1. Appuyez sur F12
  2. Onglet "Application" (Chrome) ou "Stockage" (Firefox)
  3. À gauche: Cookies → https://x.com
  4. Cherchez la ligne avec auth_token dans la colonne "Name"
  5. Copiez la valeur dans la colonne "Value"

✅ Étape 3: Ajouter au Bot

  1. Allez sur votre dashboard: http://localhost:3000
  2. Cliquez sur "New X Account"
  3. Remplissez:
    • Username: Votre username (sans @)
    • Auth Token: Collez le token copié
  4. Cliquez "Initialize Node"

✨ C'est fini! Le bot utilisera directement ce token et skippera le login!

💡 Notes Importantes

  • Le auth_token dure plusieurs mois
  • Le bot le rafraîchit automatiquement
  • Gardez-le secret - il donne accès à votre compte
  • Si les actions échouent plus tard, récupérez un nouveau token
